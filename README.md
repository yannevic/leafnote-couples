# 🌿 leafnote couples
mural colaborativo em tempo real para casais — feito com carinho 🌸

## o que é
leafnote couples é um app de casal com mural compartilhado, jardim virtual, cartas especiais, streak e muito mais. cada casal tem seu próprio espaço isolado, acessado via código de convite. desenvolvido com Electron + React + Firebase.

## funcionalidades
- 🗒️ post-its, checklists, tags, desenhos e cartinhas no mural
- 💌 cartas especiais com datas comemorativas e data de abertura
- 🌱 jardim compartilhado com sementes, estágios e modo pânico
- 📅 calendário mensal com marcação de ciclo menstrual
- 🔥 streak diário com marcos e recompensas
- 🎲 widgets: dado, timer e roleta
- 🎬 lista de filmes e séries
- 🌐 presença online em tempo real
- 📋 múltiplos murais
- 🔄 atualização automática
- 🔐 espaço isolado por casal via código de convite

## como entrar
1. baixe o instalador na página de [releases](../../releases)
2. crie sua conta com apelido, email e senha
3. um dos dois cria o cantinho e compartilha o código — o outro entra com o código
4. pronto, o mural de vocês está pronto 🌿

## stack
- Electron + React 19 + TypeScript
- Tailwind CSS v4 + Vite 6
- Firebase Realtime Database + Auth
- electron-updater

## configurar localmente
1. clone o repositório
2. copie `.env.example` para `.env` e preencha com suas credenciais do Firebase
3. instale as dependências:
```bash
npm install
```
4. rode em desenvolvimento:
```bash
npm run dev
```
5. gere o instalador:
```bash
npm run build
```

## variáveis de ambiente
veja `.env.example` para a lista completa de variáveis necessárias.
